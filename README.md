# 🎮 BG Skins

Plataforma web para compra e venda de skins de **CS2 (Counter-Strike 2)**, com foco em negociações rápidas, atendimento humanizado e inventários completos.

---

## 🌐 Sobre o projeto

O **BG Skins** é um site institucional voltado para negociação de skins, priorizando:

* 💬 Contato direto via WhatsApp
* 💎 Inventários completos e organizados
* ⚡ Experiência rápida e objetiva
* 🤝 Atendimento personalizado

Diferente de marketplaces automatizados, o projeto aposta em **interação humana** para negociação.

---

## 🚀 Funcionalidades

* 📦 Exibição de skins disponíveis
* 💬 Integração com WhatsApp para negociação
* 🎯 Interface simples e direta
* 📱 Layout responsivo
* ⚡ Carregamento rápido (site leve)

---

## 🛠️ Tecnologias utilizadas

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla)**
* Google Fonts (Inter / Outfit)

